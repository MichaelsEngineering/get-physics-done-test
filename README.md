# get-physics-done-test

Public log of using Get Physics Done with Codex for predictive control of plasma turbulence and confinement in fusion reactors.

## Goal

Use GPD as the structured research workflow layer on top of Codex to produce bounded overnight paper drafts and research artifacts.

## Stack

- Python 3.11
- uv
- Codex
- GPD

## Repo conventions

- all overnight runs go under `paper_runs/`
- all plans are committed
- all output artifacts are written to disk
- final review packet lives under `09_final/`
