## Units

### Unit System

| Field | Value |
| ----- | ----- |
| **Convention** | SI units unless a cited source requires a different convention, in which case the source convention must be stated explicitly in the artifact using it |

| **Rationale** | Keeps cross-source interpretation explicit during a literature-heavy drafting run |
| **Dependencies** | Parameter tables, quoted scalings, control relevance claims |
